============================================
Hotel Management System for CBSE class 12

Tools Used: Python, Binary Files
=============================================


This sample project is for CBSE class 12 students, it will help them to understand how simple but powerful projects
can be developed in python programming language together with its easy to use binary file handling mechanism.

About the project:

The software implements a hotel management system with all the core features like creation, editing and deleting
records of rooms and customer record;

Here is an overview of the components of this project:

main.py: It is the entry point, displays the main menu and transfers execution to other modules accordingly
myprint.py: It defines some basic functions to display data on screen properly
datafile.py: It provides common interface between the binary files in the hard disk and our program
room.py: It defines the details of a room
rooms.py: It provides all the functionalities to work with the rooms table
rooms.dat: It is the binary data file used to store the room records persistently
customer.py: It defines the details of a customer record
customers.py: It provides all the functionalities to work with the customers table
customers.dat: It is the binary data file used to store the customer records persistently
